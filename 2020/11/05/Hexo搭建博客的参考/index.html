Hexo搭建博客的参考

发表于 2020-11-05 更新于 2020-11-08 分类于 经验 阅读次数： Valine：

前两日使用Hexo搭建了自己的初步博客，过程略繁琐且曲折，浏览了很多优秀的视频教程以及帖子，对其进行了筛选并加以总结，一为帮很多像我一样的初学者省下筛选的时间，二为自己备忘，日后有问题方便查阅。

一、各个组件的关系

我觉得首先要理解好Hexo、Git、node、deploy、github等之间的关系和作用。

Git、node、hexo 的安装都是对于整个客户端来说的，所以安装之后可以在任何位置查到版本号。

但是从要进行hexo init 开始，就要注意是在此文件根目录下进行操作。

本地的博客通过 github 生成线上博客，这个从推送的桥梁就是deploy，所以deploy的安装是对于具体的文件夹的。

而要生成本地博客则与github无关。

二、参考资源

1.建立博客

从零开始建立博客推荐参考这个up的视频 B站视频

文字参考推荐知乎这篇帖子 使用 Hexo+GitHub 搭建个人免费博客教程（小白向） - 知乎

虽然视频使用Gitee部署的，但是和github无差，我一开始也是学习Gitee部署。但是不太喜欢手动刷新的环节，故推倒重来了。 重点是视频讲解很详细。 文字版的参考后来重新搭建博更快地作为参考。

2.主题与美化

yilia主题推荐这位up主的视频 yilia主题视频 基本上将主要的模块都讲解了，yilia主题本身也很优秀。我一开始也是使用的yilia主题学习的。

比较流行的next主题可以参考这个 next系列视频 这位up叔叔讲得比较慢且是对着官方文档讲解，思路清晰。虽然视频里面他犯了一个错误，弹幕比他还着急hhh，但是总体来说非常好，视频有好几期。

3.其他

其他和网上主流做法不同但是亲身体验比较好用的。

  • 浏览量统计系统valine：搜到的大多是不蒜子，纯leadcloud，但是valine(借助leancloud)就很好，且next主题里面已经配置好了相关的关键字。

  • 评论系统valine：网上的参考大多是第三方像多说什么的，但是valine自己能够both 浏览量和评论，可能是next版本更新问题。

  • 图片问题：后期写文章本地md文件能看到图片的，但是上传之后不能。在线博客只支持url，因此推荐图床和本地repository一起备份，防止图床链接失效。
